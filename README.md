## Featured Projects

### `jeu-nds`
A **Nintendo DS game project in C**, inspired by *Flappy Bird*, built to explore low-level programming and real-time interactive systems. The project includes gameplay logic, sprite rendering, collision detection, controls, and sound integration. It reflects my interest in programming closer to the hardware and understanding how embedded-style systems behave in practice.

### `EE-442_Wireless_Project`
A project in **MATLAB** focused on **OFDM system simulation** for a wireless receivers course. It allowed me to work on communication systems in a more concrete way by connecting signal processing theory with simulation and implementation.

### `project-1a-music-player`
A software project in **Kotlin** centered on a music player experience. It helped me strengthen my application development skills, especially around structuring software, managing interactions, and building practical user-oriented features.

### `Lottery Ticket Hypothesis`
As part of the **Fundamentals of Inference and Learning** course, I worked in a team of five on a project focused on reproducing results from the paper on the **Lottery Ticket Hypothesis**. This hypothesis suggests that within a large neural network, there exist much smaller subnetworks — called **winning tickets** — that, when trained with the right initial weights, can achieve performance comparable to the full model. My contribution to the project was to train the **Conv6** model and develop the pipeline used to save, organize, and display the results for all the CNN models tested in our experiments.
